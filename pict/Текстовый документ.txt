Interface - класс отвечает за распазнование команд и отправление их в Controller, также
читает config help файлы
Controller - полученные команды отправляет либо в Buffer, Field, Launch для их реализации
Launch - класс запусков методов класстерного анализа EM,k_means,Min_tree,DBScan,Wave_alg,
печатает кластеры и выдает по ним информацию
EM - выполняет EM алгоритм
k_means - выполняет алгорит к средних
Min_tree - выполняет алгоритм минимального покрывающего дерева, затем по нему строит Histogramm
DBScan - выполняет алгоритм DBScan, то есть находится нужное множество, затем с ним выполняется волновой алгоритм
Wave_alg - выполняет волновой алгоритм
Histogramm - класс, строящий гистограмму
Cluster - класс хранящий ссылки на множество точек кластера, также его центр, экстремумы, кол-во точек, 
может распечатать точки
Field - класс в котором хранится множество точек, группировка их по облакам, может распечатывать облако или поле,
добавлять облако, сохранять матрицу расстояний, находить расстояние между точками
Buffer - класс для изменения облака, загружает облако в виде множества точек, изменяет его через сдвиг, поворот,
масштабирование, выгружает в новое облако
Cloud - класс хранения множества ссылок точек в поле, может печатать облако искать экстремумы, создавать точку, хранит
экстремумы центр и дисперсии, если не из буфера
Point - класс точки хранит две координаты и значение, номер точки, может их изменять
Matrix_dist - класс хранения матрицы расстояний
Matrix - класс хранения матрицы любого размера
